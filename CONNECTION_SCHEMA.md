# Схема подключения BuhloWar (ESP32-S3)

## Общая схема

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              ESP32-S3                                       │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                        ВХОДЫ (Inputs)                                │   │
│  │                                                                      │   │
│  │   GPIO4 ────┬──── BUTTON_DOWN (Вниз)                                │   │
│  │             └──── 10kΩ ─── +3.3V                                    │   │
│  │                                                                      │   │
│  │   GPIO5 ────┬──── BUTTON_UP (Вверх)                                 │   │
│  │             └──── 10kΩ ─── +3.3V                                    │   │
│  │                                                                      │   │
│  │   GPIO6 ────┬──── BUTTON_SET (Выбор)                                │   │
│  │             └──── 10kΩ ─── +3.3V                                    │   │
│  │                                                                      │   │
│  │   GPIO7 ────┬──── BUTTON_BACK (Назад)                               │   │
│  │             └──── 10kΩ ─── +3.3V                                    │   │
│  │                                                                      │   │
│  │   GPIO3 ────┬──── DS18B20 (1-Wire шина)                             │   │
│  │             │      ├─ TSA  (ТСА - температура спирта/пара)          │   │
│  │             │      ├─ AQUA (Вода охлаждения)                        │   │
│  │             │      ├─ TSAR (Царга)                                  │   │
│  │             │      └─ TANK (Бак/куб)                                │   │
│  │             └──── 4.7kΩ ─── +3.3V                                   │   │
│  └──────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │                     I2C ШИНА (GPIO8=SDA, GPIO9=SCL)                  │   │
│  │                                                                       │   │
│  │   0x27 ─── LCD 2004 (20x4 символов, I2C конвертер)                   │   │
│  │   0x76 ─── BME280 (Темп/Влажн/Давл бокса)                            │   │
│  │   0x68 ─── DS3231 (Часы реального времени)                           │   │
│  │   0x57 ─── AT24C32 (EEPROM 32KB, на модуле DS3231)                   │   │
│  │                                                                       │   │
│  │   Схема:                                                              │   │
│  │   GPIO8 (SDA) ───┬──── SDA всех устройств                            │   │
│  │                  └──── 4.7kΩ ─── +3.3V                               │   │
│  │   GPIO9 (SCL) ───┬──── SCL всех устройств                            │   │
│  │                  └──── 4.7kΩ ─── +3.3V                               │   │
│  └───────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │                SPI для SD КАРТЫ (HSPI)                                │   │
│  │                                                                       │   │
│  │   GPIO14 ─── SCK  (Тактовый)                                         │   │
│  │   GPIO12 ─── MISO (Данные от карты)                                  │   │
│  │   GPIO13 ─── MOSI (Данные к карте)                                   │   │
│  │   GPIO15 ─── CS   (Выбор карты)                                      │   │
│  │                                                                       │   │
│  │   Карта MicroSD с файлами:                                           │   │
│  │   ├── /www/           (Web-интерфейс)                                │   │
│  │   ├── /wifi_config.txt (WiFi настройки)                              │   │
│  │   ├── /profiles/      (Профили процессов)                            │   │
│  │   └── /system.log     (Лог системы)                                  │   │
│  └───────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │                    ВЫХОДЫ 220В (через SSR/реле)                       │   │
│  │                                                                       │   │
│  │   GPIO46 ─── HEATER_1  ─── ТЭН канал 1 (рабочий)                     │   │
│  │   GPIO18 ─── HEATER_2  ─── ТЭН канал 2 (разгонный)                   │   │
│  │   GPIO42 ─── CONTACTOR  ─── Главный силовой контактор                │   │
│  │   GPIO37 ─── VALVE_HEAD ─── Клапан отбора голов                      │   │
│  │   GPIO38 ─── VALVE_BODY ─── Клапан отбора тела                       │   │
│  │   GPIO48 ─── SPARE_3    ─── Запасное реле 220В                       │   │
│  │                                                                       │   │
│  │   Все реле с опторазвязкой, тип НЗ (NC)                              │   │
│  └───────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │                    ВЫХОДЫ 12В (через SSR)                             │   │
│  │                                                                       │   │
│  │   GPIO2  ─── VALVE_WATER ─── Клапан подачи воды                      │   │
│  │   GPIO47 ─── MIXER        ─── Мешалка                                │   │
│  │   GPIO16 ─── SPARE_1      ─── Резерв SSR 12В                         │   │
│  │   GPIO17 ─── SPARE_2      ─── Резерв SSR 12В                         │   │
│  └───────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │                         ДРУГОЕ                                        │   │
│  │                                                                       │   │
│  │   GPIO21 ─── BUZZER ─── Зуммер/сигнализация                          │   │
│  └───────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Таблица пинов

### Входы

| GPIO | Функция | Тип | Примечание |
|------|---------|-----|------------|
| 3 | DS18B20 | 1-Wire | 4 датчика, pull-up 4.7kΩ |
| 4 | BUTTON_DOWN | Input | pull-up 10kΩ |
| 5 | BUTTON_UP | Input | pull-up 10kΩ |
| 6 | BUTTON_SET | Input | pull-up 10kΩ |
| 7 | BUTTON_BACK | Input | pull-up 10kΩ |
| 8 | I2C_SDA | I2C | pull-up 4.7kΩ |
| 9 | I2C_SCL | I2C | pull-up 4.7kΩ |

### Выходы 220В

| GPIO | Функция | Реле | Тип клапана |
|------|---------|------|-------------|
| 18 | HEATER_2 | SSR | НЗ (NC) - разгонный ТЭН |
| 37 | VALVE_HEAD | Реле 220В | НЗ (NC) - клапан голов |
| 38 | VALVE_BODY | Реле 220В | Настраивается (НЗ/НО) |
| 42 | CONTACTOR | SSR | НЗ (NC) - главный контактор |
| 46 | HEATER_1 | SSR | НЗ (NC) - рабочий ТЭН |
| 48 | SPARE_3 | Реле 220В | Резерв |

### Выходы 12В

| GPIO | Функция | Реле | Тип |
|------|---------|------|-----|
| 2 | VALVE_WATER | SSR | НЗ (NC) - клапан воды |
| 16 | SPARE_1 | SSR | Резерв |
| 17 | SPARE_2 | SSR | Резерв |
| 47 | MIXER | SSR | НЗ (NC) - мешалка |

### SPI (SD карта)

| GPIO | Функция | SPI |
|------|---------|-----|
| 12 | MISO | HSPI |
| 13 | MOSI | HSPI |
| 14 | SCK | HSPI |
| 15 | CS | HSPI |

### Прочее

| GPIO | Функция | Примечание |
|------|---------|------------|
| 21 | BUZZER | Зуммер |

## I2C Устройства

| Адрес | Устройство | Функция |
|-------|------------|---------|
| 0x27 | LCD 2004 | Дисплей 20x4 |
| 0x76 | BME280 | Датчик бокса (T/H/P) |
| 0x68 | DS3231 | Часы реального времени |
| 0x57 | AT24C32 | EEPROM 32KB |

---

## Датчики DS18B20

| Индекс | Назначение | Описание |
|--------|------------|----------|
| 0 | TSA | Температура спирта/пара |
| 1 | AQUA | Температура воды охлаждения |
| 2 | TSAR | Температура царги |
| 3 | TANK | Температура бака/куба |

---

## Типы клапанов

| Клапан | По умолчанию | Настройка |
|--------|--------------|-----------|
| VALVE_HEAD (Головы) | НЗ (NC) | Фиксировано |
| VALVE_BODY (Тело) | НЗ (NC) | Настраивается в меню |
| VALVE_WATER (Вода) | НЗ (NC) | Фиксировано |

**НЗ (NC) = Нормально Закрыт**: без питания клапан закрыт

## Схема питания

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   220В AC   │────▶│  БП 12В DC  │────▶│   ESP32-S3  │
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ├──▶ SSD 12В (4 шт.)
                           └──▶ Доп. устройства 12В

┌─────────────┐     ┌─────────────┐
│   220В AC   │────▶│ SSR 220В    │────▶ ТЭН, Контактор
└─────────────┘     └─────────────┘

┌─────────────┐     ┌─────────────┐
│   220В AC   │────▶│ Реле 220В   │────▶ Клапаны 220В
└─────────────┘     └─────────────┘
```
